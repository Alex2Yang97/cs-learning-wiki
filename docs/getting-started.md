# Getting Started

A sample page. Delete or replace it with your own notes.

## Local preview

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
mkdocs serve
```

Then open <http://127.0.0.1:8000> — the site live-reloads as you edit.

## Useful commands

| Command          | What it does                          |
| ---------------- | ------------------------------------- |
| `mkdocs serve`   | Local dev server with live reload     |
| `mkdocs build`   | Build the static site into `site/`    |
